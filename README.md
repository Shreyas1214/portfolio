## 📖 About This Project

This repository contains the source code for my professional cybersecurity portfolio. It was explicitly designed to break away from the typical "hacker/terminal" stereotypes often seen in the infosec community. Instead, it utilizes a highly refined, corporate-grade aesthetic—focusing on heavy whitespace, precise typography, and subtle micro-interactions to present technical achievements with executive-level polish.

The site is a **Dependency-Free Frontend**, meaning it avoids heavy frameworks like React or Next.js in favor of Vanilla TypeScript and CSS, compiled blazingly fast using Vite.

## ✨ Key Features

- **Data-Driven Architecture:** All professional experience, skills, and certifications are decoupled from the UI and driven by a central `data.js` structure, making content updates effortless.
- **Modern UI/UX:** Clean typography (Inter), glassmorphism cards, and sleek gradient accents create a premium feel.
- **Zero-Image Animations:** 3D parallax effects and floating elements are built entirely using infinite CSS animations and inline SVGs to ensure a perfect Lighthouse performance score.
- **Scroll Reveal Dynamics:** Content elegantly fades and slides into view via a custom IntersectionObserver implementation.
- **Responsive Layouts:** A mobile-first CSS Grid and Flexbox foundation ensures flawless rendering across all device sizes.

## 🏆 Highlight Sections

- **Impact & Metrics:** Front-and-center display of key career achievements (e.g., 95% phishing reduction, 40% faster incident response).
- **Hall of Fame & Recognitions:** Detailed showcase of responsible disclosure acknowledgments from **India's NCIIPC**, **Wageningen University**, **PayPal**, and others.
- **Bug Bounty Metrics:** Real-time reflection of 50+ websites secured and multiple security badges.
- **Interactive Expertise:** 73+ categorized technical skills with hover states.

## 🚀 Local Development

To run this project locally:

```bash
# 1. Clone the repository
git clone https://github.com/Shreyas1214/portfolio.git
cd portfolio

# 2. Install development dependencies
npm install

# 3. Start the Vite development server
npm run dev

# 4. Build for production
npm run build
```

## 📦 Deployment Pipeline

This repository is configured with a continuous deployment workflow. Any changes pushed to the `main` branch are automatically built via Vite and deployed to GitHub Pages using GitHub Actions.
